# Portfolio DevOps : Architecture Microservices K8s

Ce projet démontre une chaîne DevOps complète (End-to-End) avec :
- **Backend :** Java Spring Boot
- **Frontend :** Next.js
- **Infrastructure :** Docker, Kubernetes
- **CI/CD :** GitHub Actions (à venir)

## Structure
- `/backend` : API REST
- `/frontend` : Interface Utilisateur
- `/k8s` : Manifestes Kubernetes
